CSS Logical Properties

Logical properties

Logical properties are a way that we can declare things like margin, padding, borders and even text alignment, which can be set based on the language used, rather than having to redefine it if the language has a different writing direction.

الخصائص المنطقية

الخصائص المنطقية هي طريقة يمكننا من خلالها إعلان أشياء مثل الهامش ، والحشو ، والحدود ، وحتى محاذاة النص ، والتي يمكن تعيينها بناءً على اللغة المستخدمة ، بدلاً من الاضطرار إلى إعادة تعريفها إذا كان للغة اتجاه كتابة مختلف.

論理プロパティ

論理プロパティは、マージン、パディング、ボーダー、さらにはテキストの配置などを宣言できる方法であり、言語の書き込み方向が異なる場合に再定義するのではなく、使用する言語に基づいて設定できます。
